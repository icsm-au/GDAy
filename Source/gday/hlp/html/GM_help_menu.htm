Help menu

This menu contains four items; three for displaying the online help documentation and one to display the about box. The current online help documentation utilises the latest version of Microsoft's HTML Help Viewer.

Contents
Select this menu item to display the table of contents for the online help documentation.
Search
Select this menu item to search through the online help documentation for particular contents topics, page headings or key words.
Index
Select this menu item to display the Index to the online help documentation.
About
Select this item to view information about the version, copyright and general software information.

 

Notes

  • Anytime when using GDAy, you can access this help documentation by pressing F1.
  • For Help related to a specific dialog, click on the available Help button or press F1.
