Tools menu

This menu contains two items for selecting the input and output coordinate parameters and user (custom) settings for the entire GDAy interface respectively.

Coordinate System
This menu item opens the Coordinate System dialog box. Here, you can select the input and output coordinate parameters for interactive, points file and folder transformations.
Options
Options directs the user to a 3-page property dialog box that allows you customise the GDAy user interface, interactive output file and log file settings. Select this option to modify the user settings for point numbering, auto format of edit boxes, working precision of output edit boxes, station naming, default file locations for the interactive output and log files, and header text information.
View Log File
This menu option opens the default log file NotePad.

Note

  • The Tools menu can also be accessed by right-clicking on the GDAy dialog.
