To extract part of a grid using coordinates that define an area or location

  1. Run Grid Extractor and then click browse to open the specified source grid file.
  2. Click browse to specify the location of the new grid file name.
  3. Enter in the required grid limits.
  4. Click Extract.

 

Note

  • For more information about how to create a new grid using Grid Extractor, open the Grid Extractor online documentation to view the on line Help Table of Contents or click Help on the Grid Extractor dialog.
