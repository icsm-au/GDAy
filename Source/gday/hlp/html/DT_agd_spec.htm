Definition of the Australian Geodetic Datum (AGD)

As proclaimed in the Australian Commonwealth Gazette of 6 October 1966, the Australian Geodetic Datum was defined by the following parameters:

Reference ellipsoid
  • Australian National Spheroid (ANS):  a best fitting local ellipsoid for the geoid in the Australian region.
  • Fundamental parameters: 
    Semi-major axis (a):  6,378,160.0 metres
    Semi-minor axis (b):  6,356,774.719 metres
    Flattening (f):  1/298.25
    • Origin
  • Johnston Geodetic Station: located in the Northern Territory
  • Coordinates: 
    Latitude:  S 25° 56' 54.5515"
    Longitude:  E 133° 12' 30.0771"
    Ellipsoidal ht:  571.2 m

    Coordinate sets produced on the AGD

    Australian Geodetic Datum 1966 (AGD66)
  • The fundamental set of geographic coordinates produced by the least squares adjustment of the Australian geodetic network in March 1966, using the Australian Geodetic Datum.
    • Australian Map Grid 1966 (AMG66)
  • The resultant grid, or projection coordinates , derived from a Universal Transverse Mercator projection of the AGD66 coordinate set, using the ANS.
    • Australian Geodetic Datum 1984 (AGD84)
  • The coordinate set produced following the re-adjustment of the Australian geodetic network in 1982 (known as the Geodetic Model of Australia 1982), which included the original 1966 adjustment and additional terrestrial and satellite observations. The adjustment provided an improved measure of accuracy for Queensland, South Australian and Western Australia however, did not significantly improve the AGD66 coordinate set for the remaining states. Therefore, the adopted coordinate set for the Northern Territory, Australian Capital Territory, New South Wales and Victoria remained as the AGD66.
    • Australian Map Grid 1984 (AMG84)
  • The resultant grid coordinates derived from a Universal Transverse Mercator projection of the AGD84 coordinate set, using the ANS.
