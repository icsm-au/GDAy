Conformal transformation (7 Parameter) formula

The formula used by GDAy to transform coordinates on one datum to another using a conformal transformation method is the Bursa Wolf model. This method of transformation is related specifically to the reference frame from which the coordinates were derived and is computed in terms of earth-centred, cartesian coordinates of (X, Y, Z). The relationship is developed as a set of 7 parameters: 3 origin translation elements (DX, DY, DZ), 3 rotations for the coordinate axes (RX, RY, RZ), and a scale factor. The Bursa Wolf model shown above is appropriate for small rotation angles (ie. less than 5 seconds of arc) in the coordinate axes and is generally applicable to coordinate transformations between reference ellipsoids.

 

Important

  • Before you use the above formula, ensure that you are using the appropriate transformation parameters.

Notes

  • These parameters can be used for projects of medium accuracy (of the order 1 m). The Grid Transformation method must be used for projects requiring greater accuracy.
  • National parameters to convert between GDA94 and GDA2020 coordinate sets are available in the GDA Technical Manual(http://www.anzlic.org.au/icsm/gdatm/).
  • To transform geographic coordinates using this method, the coordinates must be transformed to cartesian coordinates. For more information, see Cartesian to geographic (earth centred XYZ to spheroid) formula.
