To select the distortion grid file

  1. Point to Distortion Grid Settings on the File menu, and then click Select New Grid File.
  2. Click Browse.
  3. In the Look in box, click the drive or folder that contains the distortion grid file.
  4. In the Files of type box, choose the appropriate grid file-type or format (note: the available types include ASCII (*.asc) and Binary (*.gsb) files).
  5. In the folder list, double-click folders until you open the folder that contains the file.
  6. Double-click the distortion grid you want to use.

 

Notes

  • For more help on selecting a distortion grid and transformation parameters, see Choosing a distortion grid file.
  • To obtain the latest version of the national GDA94 distortion grid, download it from the ICSM GitHub Site.
  • If you experience any difficulties in obtaining the grids, contact the ICSM.
  • If there is no distortion grid for your area, you may still perform 7-Parameter transformations (medium accuracy ~ of the order 1 m) on the main GDAy dialog.

Transformation Parameters

Whenever you are transforming a single point or a file of points and the point being transformed lies outside the limits of the distortion grid, a 7-Parameter datum transformation (also known as a 3-Dimensional similarity transformation) may be performed. In this case, it is important that you select the most appropriate transformation parameters for your area of interest. For more information, see Conformal transformation (7 Parameter) formula.

  1. Point to Distortion Grid Settings on the File menu, and then click Select New Grid File.
  2. From the Parameters drop-down combo box, click on the desired transformation parameters.
  3. Click OK to accept the changes.

 

Note

  • These parameters can be used for projects of medium accuracy (of the order 1 m). To achieve the greatest accuracy, the transformation of a point must incorporate the distortion grid.
  • When you have selected a new distortion grid, the filename is displayed in the status bar on the bottom of the main GDAy dialog.
  • If an invalid grid file is chosen, or the grid file is corrupt, then you will be prompted to continue with a conformal 7-Parameter coordinate transformation each time a transformation is performed.
  • Although this method of coordinate transformation requires a valid ellipsoidal height, you are not prompted to enter in any such value. Where a 7-Parameter coordinate transformation is performed, an ellipsoidal height of zero (0) is used by default.
