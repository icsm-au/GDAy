To change the default path of the log file

  1. On the Tools menu, click Options and then click the Log File tab.
  2. In the File settings group, click Change.
  3. In the Save in box, click the drive or folder that you want to save the file in.
  4. In the Save as type box, note that the only predefined file-type is GDAy Log Files (*.log).
  5. In the folder list, double-click folders until you open the folder that contains the file. To create a new folder, click the Create New Folder icon and then type a name for the new folder in the Name box.
  6. In the file name box, type in the name of the file you want to save as the default log file. Otherwise, select an existing *.log file from the list to be overwritten the next time you start GDAy.
  7. Click OK.

 

Notes

  • If you change the Default file name and then click OK to accept the Log File options, a new log file is created.
  • You can create a new log file at any time when using GDAy.
  • You can view the contents of the log file at any time when using GDAy by clicking View from the Log File tab.
  • You can add a personalised header to your log file by adding information into the Add header to log file text box.
