Managing global user preferences

When using GDAy, you can choose between a variety of options that customise the display of information on the main GDAy dialog, the recording of coordinate transformations, and the logging of other essential information.

Using the Options property sheets, the following three options are available when using GDAy:

User interface options

Using the View tab, you can opt to use point numbering for each point to be transformed, format the display of coordinates in the input edit boxes, and specify the output working precision on the main GDAy dialog.

Additional information

Interactive output file options 

Using the Interactive Output tab, you can change the default file settings for the interactive output file and include station naming to all points printed to the interactive output file.

Additional information

Log file options

Using the Log File tab, you can change the default file location and customise the header for the log file.

Additional information

 

Note

  • To open the Options property sheets, click Tools and then click Options
