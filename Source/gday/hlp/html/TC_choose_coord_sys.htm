To select the right coordinate system parameters 
  1. On the Tools menu, click Coordinate System.
  2. From the drop-down combo box in the Input Coordinates group, click on the desired datum. If a new datum was selected, the input Coordinates should change to reflect the relevant types. Select the desired coordinates to be used.
  3. If you selected geographic, you now have the option to work with degrees, minutes and seconds; degrees and minutes, or; decimal degrees format (as given by the radio buttons). Otherwise, the three radio buttons will become greyed. This indicates that the coordinates are projected and that they will be displayed in metres.
  4. Follow the same procedure above for the Output Coordinates group.
  5. If your output coordinates will be projected and you would like to force the coordinates onto another zone, click the Prompt for output zone check box.
  6. Click OK to accept the changes.

 

Notes

  • You may work with any possible combination of coordinate and datum settings.
  • The selected coordinate system settings apply to both interactive and points file transformations.
  • If the input and output datum settings are the same, then a datum transformation will not occur; only a conversion of coordinates will be performed. If Cancel is pressed, the original coordinate system parameters are retained.
  • After closing and running GDAy at another time, the parameters you set previously will be retained.
  • The Tools menu can also be accessed by right-clicking on the GDAy dialog.
