Comma separated values files (*.csv)

Every record (or line) in a comma separated file must contain data fields separated by commas. You may however, omit certain fields depending on what they are. If you choose not to enter in the non-compulsory fields, you must still enter in the commas to delineate the presence of the particular field. Below is a description of the compulsory and non-compulsory fields in the required order:

FieldCompulsory?
Point IDNo
Latitude / EastingYes
Longitude / NorthingYes
Orthometric Height (AHD)No
UTM zoneYes (only when using projection coordinates)
Datum (G94 or G20)No
DescriptionNo

Decimal Degrees:

STATION B,-26.926733461,149.654651687,,55,,1st Order Control

DMS HP Notation:

STATION B,-26.553624046,149.391674607,,,,1st Order Control

Separated DMS Fields:

STATION B,-26,55,36.24046,149,39,16.74607,,,,1st Order Control

DMIN HP Notation:

STATION B,-26.556040077,149.392791012,,,,1st Order Control

Separated DMIN Fields:

STATION B,-26,55.6040077,149,39.2791012,,,,1st Order Control

Projection Coordinates

STATION B,763609.224,7018902.575,,55,,1st Order Control

 

Notes

  • From the Sample Files folder where GDAy was installed, open a *.csv file that matches the file type you are working with. Copy the first line and paste this line into your new file to help you format your data using the correct number of commas.
  • The UTM zone is only required when you are transforming a file of projection coordinates.
  • When you are entering in geographic coordinates in separated DMS fields, you must include a comma to separate the degrees, minutes and seconds values. Similarly for separated DMIN fields, you must include a comma to separate the degrees and minutes values.
  • When a certain data field is not required, you must still enter in the commas to delineate the presence of the field. If a record in your input file contains less than the required number of commas (ie. fields), the record is omitted from the output file and printed to the log file.
  • The simplest way to create a comma separated file of your own is to enter the required data in Microsoft Excel. For example, enter in the Point ID into column A, Latitude / Easting into column B, and so on (leaving the non-compulsory cells blank if not desired), then save it as a Comma separated file (*.csv).
