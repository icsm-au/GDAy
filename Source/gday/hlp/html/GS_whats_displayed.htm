What's displayed in the main GDAy dialog

At the top-left of the dialog is the main menu. You can use this menu to select the appropriate distortion grid, transform files of coordinates, select the input and output coordinate system parameters, customise the user interface settings, and to view the on-line help documentation. For more information on the available menu items, select one of the GDAy Menu help topics.

The remaining control items on the dialog are for transforming coordinates for a single point. The edit box at the top left is for point numbering. Since you can have all transformed points printed to an interactive output file, numbering each point enables you to uniquely identify them from a list. The use of point numbering is not essential for the transformation of coordinates and can be turned on or off at any time. For more information on the options for numbering each point you transform, see the help information on Show point numbering.

The input and output coordinate edit boxes are for entering in the desired coordinates and displaying the transformed values respectively. You can work in geographic coordinates or projection coordinates. The transformed (output) coordinates can be displayed to varying precision relevant to your use. The edit boxes on the far right are used to display the interpolation accuracy derived from the transformation process. When transformations are performed through the edit boxes on this dialog, the action is known as an interactive transformation.

The information relating to datum and coordinates displayed in the input and output group boxes is provided to keep the user informed of the current coordinate system parameters. This information reflects the selection made from the coordinate system parameters dialog box. To modify the desired coordinate system parameters, see the help information on Choosing the right coordinate system parameters.

The status bar pane on the bottom of the dialog displays the name of the default distortion grid file and the interactive output file output file.
