To view the current distortion grid’s properties
  1. On the File menu, point to Distortion Grid Settings and then select Current Grid Properties. This will display the Distortion Grid Properties dialog box.

 

Following is a summary of the information displayed in the Distortion Grid Properties dialog box.

FeatureDescription
File nameThe full file path.
File typeASCII or Binary. (Note. “Binary (Australian)” will be shown if you have selected the distortion grid that was initially released for your jurisdiction.
VersionThe file version. The latest version of the Queensland distortion grid is “QLD_0900”. The previous version of the Queensland grid released with GDAy 2.0 was “QLD_1299”.
No. Sub-GridsA distortion grid may contain several sub grids of different densities. The limits of these sub grids will all be contained within the parent grid. If there is only one sub grid, then it is the parent grid.
Sub-Grid NameThis drop-down box shows all sub-grids within the parent grid. Depending on the number of sub grids, you may view the limits for each sub grid. The North, South, East and West limits will be displayed for the selected sub grid name.
Reference SystemsThe reference systems for which the distortion grid supports coordinate transformations from and to. For example, the reference system for the Geocentric Datum of Australia (GDA) is the Global Reference System of 1980 (GRS80).

 

Notes

  • The current grid filename is displayed in the status bar on the bottom of the main GDAy dialog.
  • If a grid file has not been selected, you will not be able to view the Distortion Grid Properties dialog box.
