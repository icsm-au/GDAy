To control the auto format of geographic coordinates in the the input and output edit boxes

  1. On the Tools menu, click Options and then click the View tab.
  2. Do one or more of the following:
    • Select the Latitudes as “positive south” check box if you want GDAy to recognise both positive and negative latitude values as being in the Southern Hemisphere. This option does not apply for file transformations and as such, you must use negative values for latitudes in the Southern Hemisphere when creating a file of coordinates.
    • Select the Geographic values with “N/S” and “E/W” check box if you want GDAy to format the input and output coordinate values with characters to reflect their position in relation to the equator and Greenwich meridian. For example, if –27.24373345 is entered in for latitude and 153.57386347 is entered in for longitude, when you press the Tab key the edit boxes will show S 27.24373345 and E 153.57386347 respectively.
    • Select the Degrees minutes and seconds values with symbols check box if you want GDAy to display symbols degrees minutes and seconds. For example, if –27.24373345 is entered in for latitude and 153.57386347 is entered in for longitude, when you press the Tab key the edit boxes will show … 27° 24’ 37.3345” and … 153° 57’ 38.6347” respectively. This also applies if your input or output coordinates are in degrees and minutes or decimal degrees.

 

Notes

  • This auto format option applies only to geographic coordinates displayed in the input and output edit boxes on the main GDAy dialog.
  • To use the auto format option correctly, only enter in numeric characters in the input edit boxes.
  • Each time you tab out of an input edit box, the auto formatting re-applied.
  • If you are working with projection coordinates, your input coordinates will not be formatted.
