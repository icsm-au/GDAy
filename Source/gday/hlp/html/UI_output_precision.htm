To set the precision of the  coordinates on the main GDAy dialog

  1. On the Tools menu, click Options and then click the View tab.
  2. From the Working precision group, do one or more of the following:
    • In the Geographic edit box, specify the number of decimal places for the output geographic coordinates.
    • In the Projection edit box, specify the number of decimal places for the output projection coordinates.

 

Notes

  • The Working precision option applies only to the values in the Output coordinates edit boxes on the main GDAy dialog.
  • You can view up to 7 decimal places for seconds and 4 decimal places for metres.
  • Where the output coordinates are in decimal degrees, the displayed precision will be the number of decimal places for seconds plus 4. For example, if you change the value in the Minutes / Seconds: edit box to 6, each time you press Compute (depending on the type of output coordinates you have selected) the output edit boxes will show
