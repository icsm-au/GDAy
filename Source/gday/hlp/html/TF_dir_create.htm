Creating a new folder

When you are transforming folders of files and you have the Search for files in all subfolders option selected, GDAy searches the output folder for subfolders having the same directory structure as found in the input folder. If a subfolder from the input folder is not found in the output folder, you will be prompted to proceed with creating a new subfolder.

For example, if your input folder, called 'Data', contains a subfolder called 'GDA_ddeg' and the output folder does not contain a folder of this name, you will be prompted to create <output folder>\ GDA_ddeg. When prompted, you will have five options to choose from, with an additional option to access to the On-Line Help. Below is a description of the five optional buttons.

Button Action
Yes Create this subfolder only. Show this prompt again when another subfolder is not found
Yes All Create any subfolders not found in the output folder without prompting
No Skip the creation of this subfolder only. Show this prompt again when another subfolder is not found. If this subfolder contains other subfolders, they will be ignored
No All Skip the creation of any subfolders not found in the output folder. If any subfolders found contain other subfolders, they will be ignored also
Cancel (Esc) Exit the folder transformation

 

If you specify the output folder to be the input folder, the output files are given filenames that resemble the respective input files with a "_" prefix. In this instance, the output files will be created throughout the entire folder-subfolder structure and you will not be prompted to create any new folders.

 

Note

  • You will not be prompted to create a new folder if you have not selected the Search for files in all subfolders option.
