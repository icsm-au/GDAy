File menu

The file menu contains five main items to select from.

Distortion Grid Settings
Distortion Grid Settings is a pop-up menu that contains four lower level menu items:
 
  • Select New Grid File opens the Select Distortion Grid dialog box. This dialog is used to select the default distortion grid and the transformation parameters to be used only when a point lies outside the limits of the selected distortion grid. The only supported NTv2 file types are ASCII (*.asc) and Binary (*.gsb) files.
  • Grid Properties to view general and technical information about the current distortion grid.
  • Export Grid File contains two lower level menu items: ...to ASCII and ...to Binary. Depending on the file type of the currently selected distortion grid, you can export the grid file to either ASCII or Binary in the NTv2 file format. For example, if you have selected a Binary grid file, you will only be able to export it to ASCII format. Selecting either menu item displays a standard file-open dialog box from which the user can select the desired distortion grid file name. The only supported NTv2 file types are ASCII (*.asc) and Binary (*.gsb) files.
  • The Run Grid Extractor menu option activates the Grid Extractor Tool. The Grid Extractor option is available only if you selected to install Grid Extractor during Setup.
    • Interactive Input
    Interactive Input is a pop-up menu that contains two lower level menu items:
     
  • New Output File creates a new interactive output file to store the output of all subsequent interactive transformations.
  • View Output File opens the current output file in NotePad.
    • Transform Points File
    This menu item opens the Points File transformation dialog box. This dialog is used to transform coordinates contained in files of recognised format.
    Transform Folder
    This menu item opens the Folder Transformation dialog box. This dialog is used to transform an entire folder, including any subfolders, containing files of a specified type.
    Exit

    As with most other Windows-standard applications, Exit closes GDAy.

    Note

    • The Distortion Grid Settings sub menu and Interactive Output sub menu can also be accessed by right-clicking on the respective status bar pane.
