To modify the log file header

  1. On the Tools menu, click Options and then click the Log File tab.
  2. In the Add header to log file text box, add the information relevant to your needs.
  3. Wherever you type in << Date >> (exactly, with only one space either side of the word “Date”), the full system date at the time of creating the file will be printed to the log file. For example, if the following is typed into the text box:

    /////////// GDAy Transformation Log File ///////////

    Created: << Date >>

    Version 3.0

    ///////////////////////////////////////////////////////////////////////////

     

    the following will be printed to the log file:

    ///////////            GDAy Transformation Log File             ///////////

Created: Tuesday, 26 March 2002, 01:12:13 PM. 

Version 3.0

    ///////////////////////////////////////////////////////////////////////////

 

Notes

  • Note that the text will not appear exactly in the log file as it does in the text box.

  • You can enter in a maximum of 500 printable characters.
  • To place a “newline” character, put the cursor where you want it to be in the text box and press Ctrl + Enter.
  • Your data will still be printed correctly even if it extends past the width of the text box.
