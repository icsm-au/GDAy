Transform menu

The Transform menu contains two menu items to select from: GDA94 to GDA2020 and GDA2020 to GDA94. These items duplicate the functionality of the Compute button and are available only when the respective input and output coordinate system parameters match the menu item's caption.

For example, if the input coordinates are in GDA94 and the output coordinates are in GDA2020, only the first menu item (GDA94 to GDA2020) is available. Transforming coordinates in this direction is known as a forward transformation. Therefore, where a reverse transformation is required, the second menu item is only available when the input and output parameters are GDA2020 and GDA94 respectively.
